# HeadCount 2.0

HeadCount 2.0 is a project designed to challenge your skills at manipulating data and creating small, reusable React components.  

## Steps to run App

$ git clone https://github.com/chrisboylen/headcount2.0.git
$ cd headcount2.0
$ npm i
$ npm start
